## Portfolio Website

This is a simple personal portfolio website created using **HTML** and **basic CSS**.  
The project is designed to showcase a clean layout, basic styling, and fundamental web development concepts.

## Pages Included

- **Home Page** – Introduces the portfolio and provides a brief overview.
- **Contact Page** – Contains contact information and a simple way to reach out.

## Technologies Used

- HTML 
- CSS 

##How to view

- Clean and minimal design  
- Easy navigation between pages  
- Beginner-friendly structure  
- Fully static website  

## Purpose

This project was built to practice core HTML and CSS concepts and to create a basic portfolio website suitable for beginners.

## How to Use

1. Clone the repository  
2. Open `index.html` in your browser  
3. Navigate between the Home and Contact pages
